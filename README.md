# homebrew-tap

## How do I install these formulae?
`brew install tgymnich/tap/<formula>`

Or `brew tap tgymnich/tap` and then `brew install <formula>`.

## Formula

- hdf5-mpich
- cubicsdr
- soapyaudio
- soapyremote
- soapymultisdr

## Documentation
`brew help`, `man brew` or check [Homebrew's documentation](https://docs.brew.sh).
